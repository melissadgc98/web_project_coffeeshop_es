# Página Web para Coffee Shop Triple Espresso

Descripción: Este proyecto crea una página web para la cafetería de Triple Ten llamanda Triple Espresso. La página web está enfocada principalmente en añadir un método de reservar mesas en la cafetería, pero también incluye información sobre horarios, dirección y contactos.

Fucionalidad: La creación de la página web se realizó utilizando el lenguaje html de marcado para la estructura de la página. Para el diseño de la página se utilizó el lenguaje de estilos CSS. Mediante técnicas avanzadas se desarrollo un formulario que permite al usuario ingresar sus datos y la información para realizar una reserva.

Mejoras del proyecto: Este proyecto aun cuenta con limitantes,

1.  Aun no se tiene definido el método de registro de las reservas.
2.  Los links a los contactos no dirigen a una página existente.
3.  No hay método de confirmación de la reserva para el cliente (página de confirmación, aviso en algún canal como email o mensaje)
4.  Los horarios de reserva no están limitados a los horarios de atención que se establecen en los detalles.

Para ofrecer un correcto funcionamiento de la página web se necesitaría al menos desarrollar los puntos sugeridos anteriormente.
